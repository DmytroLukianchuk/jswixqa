Занятие 2. Задания

Создайте страницу index.html, при открытии которой в консоль выведется с помощью console.log результат таких выражений:

  1. Дан массив: ['jazz', 'rock', 'pop', 'r&b', 'lounge'], необходимо вывести в консоль true если массив содержит 'jazz' и false — если такой элемент отсутствует.

    Check the console for the result

  2. Дан объект:
var site = {
    pages: [
    {
    id: 1,

    url: '/'
    },
    {
    id: 2,
    name: 'Contact Page',
    url: '/contact'
    },
    {
    id: 3,
    name: 'About Page',
    url: '/about'
    },
    {
    id: 4,
    name: 'Photo Page',
    url: '/catalog',
    photos: [
    {
    name: 'holidays',
    url: '/photos/holiday.jpg'
    },
    {
    name: 'Egupt',
    url: '/photos/egupt.jpg'
    },
    {
    name: 'Germany',
    url: '/photos/germany.jpg'
    }
    ]
    },
    ]
    }

Необходимо вывести в консоль название последней фотографии из страницы "Photos" из данного объекта, т.е. 'Germany', построив правильный запрос к нужному свойству.
